# Dictionary

## Giới thiệu
Đây là ứng dụng cá nhân hỗ trợ: Lưu và tra cứu từ vựng cá nhân có dịch từ tiếng anh sang việt và ngược lại.

---

### Cách sử dụng:
- Nhập từ mới vào tệp **dictionary.txt**(ví dụ:
  run | chạy | V )
### Cách chạy ứng dụng:
- Chạy tệp **quiz.exe** để bắt đầu sử dụng thư viện.
